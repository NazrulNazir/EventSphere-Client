# 🎉 EventSphere

EventSphere is a full-stack event management platform built with **Next.js**, **TypeScript**, **Node.js**, **Express.js**, and **MongoDB**. Users can browse events, create new events, update their own events, and manage them through a personalized dashboard.

## 🚀 Features

* Authentication & Authorization
* Create, Update & Delete Events
* View All Events
* Event Details Page
* User Dashboard
* Light & Dark Theme
* Responsive Design
* Secure API with MongoDB

## 🛠️ Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS
* shadcn/ui
* Better Auth

### Backend

* Node.js
* Express.js
* MongoDB

## 📦 Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Create a `.env` file and add your environment variables.

Run the project

```bash
npm run dev
```
